# Question

Given the following code, what would appear in the console?&nbsp;

&nbsp;

```
function Parent() {
  const header = "Hello"
  const contents = "World"
  
  return <child title="{header}" body="{contents}" minutestoread="{10}"></child>
}

function Child(props) {
  console.log(props);
  // ... return JSX
}
```

## Correct

```
{
  title: "Hello",
  body: "World",
  minutesToRead: 10
}
```

## Incorrect

```
{
  header: "Hello",
  contents: "World",
  minutesToRead: 10
}
```
